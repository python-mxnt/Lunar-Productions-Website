# Lunar Productions Website

Welcome to the official website repository for Lunar Productions, a creative team specializing in various forms of digital art and media production.

## About Lunar Productions

We are a creative team passionate about producing high-quality content across multiple mediums:

- **Editing**: Specializing in video editing using professional software like DaVinci Resolve, Capcut (desktop), and Blender
- **Videography**: Capturing moments and bringing ideas to life through film
- **Photography**: Creating stunning visual content (work in progress)

## Getting Started

To view the website locally:

1. Clone this repository
2. Open `Frontend/index.html` in your preferred web browser
3. Navigate through the site using the top navigation bar

## Features

- Responsive design for various screen sizes
- Clean, modern interface
- Easy navigation between pages
- Contact form for inquiries and feedback
